📌 Overview

BGMI Scrims Management System is a web-based platform designed for esports organizers to efficiently manage scrims and tournaments.

It simplifies:

Team registrations
Slot allocation
Match scheduling
Admin control

Built with a focus on performance, simplicity, and scalability.

✨ Key Features
👤 User Features
Join available scrims
Real-time slot tracking
Team registration system
View match details & schedule
🛠️ Admin Features
Create, update, and delete scrims
Control registration (open/close anytime)
Manage teams and bookings
Approve or reject participants
Set entry fee and prize pool
⚙️ Core Functionality
Automatic slot management system
Booking status tracking (approved/pending)
Clean and responsive UI
Optimized database queries
🧱 Tech Stack
Layer	Technology
Frontend	HTML, CSS, JavaScript
Backend	PHP
Database	MySQL (phpMyAdmin)
Server	XAMPP / Apache

⚙️ Setup & Installation
1. Clone Repository
git clone https://github.com/YOUR_USERNAME/bgmi_scrims_management.git
2. Move Project

Copy the folder to:

C:\xampp\htdocs\
3. Start Server
Start Apache
Start MySQL
4. Setup Database
Open phpMyAdmin
Create database: bgmi_scrims
Import SQL file from /database
5. Configure Connection

Edit:

config/db.php
$host = "localhost";
$user = "root";
$password = "";
$database = "bgmi_scrims";
6. Run Project
http://localhost/bgmi_scrims_management
