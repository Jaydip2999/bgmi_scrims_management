🔥 About Project

A powerful and modern web application to manage BGMI Scrims & Tournaments with ease.
Designed especially for scrim organizers to automate team registrations, slot management, and match handling.

✨ Features
👥 User Panel
✅ Join Scrims Easily
✅ Real-time Slot Availability
✅ View Match Details
✅ Team Registration System
🛠️ Admin Panel
🔥 Create / Manage Scrims
🔥 Start / Stop Registration
🔥 Set Entry Fee & Prize Pool
🔥 Approve / Reject Teams
🔥 Auto Slot Full Detection
⚡ Core System
💡 Dynamic Slot Management
📱 Fully Responsive Design
🎨 Clean UI/UX
⚡ Fast Performance
